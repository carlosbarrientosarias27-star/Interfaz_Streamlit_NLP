# Guía de Uso: Interfaz Streamlit

Este manual describe cómo operar, configurar y personalizar la interfaz de usuario (UI) del proyecto **INTERFAZ_STREAMLIT_NLP**.

## 1. Requisitos Previos

Antes de ejecutar la interfaz, asegúrate de cumplir con lo siguiente:
* **Ollama:** El servidor debe estar corriendo localmente (`ollama serve`).
* **Dependencias:** Haber instalado los paquetes de `requirements.txt`.
* **Directorio:** Situarse en la raíz del proyecto.

## 2. Ejecución de la Aplicación

Para iniciar el servidor de Streamlit y abrir la interfaz en tu navegador, ejecuta:

```bash
streamlit run interface/demo_app.py
